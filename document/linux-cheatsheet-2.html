Understanding File Paths: Absolute vs. Relative

This is a core concept for command line work. A file path tells a command where to find or create something on the computer.

  • Absolute Paths: Start at the root directory (/) and provide the full, specific location of a file or folder. They work from anywhere on the computer.
    Example: /home/sandbox/desktop/new_folder
  • Relative Paths: Start from your current working directory. They do not begin with a /. They are shorter and easier to type, but they only work if you are in the correct starting location.
    Example: If you are already in /home/sandbox, you could use desktop/new_folder to create a folder there.

Essential Commands ‚å®Ô∏è

Command Purpose Example & Notes
mkdir Make directory. Creates a new folder. Accepts a relative or absolute path as an argument. mkdir my-stuff (creates a new folder named my-stuff in your current location)
mkdir /home/sandbox/new-folder (creates a new folder using the full path)
mv Move. Used for "cutting and pasting" a file or folder from one location to another. It takes at least two arguments: the item you want to move and the destination. mv file1 folder1/ (moves file1 into folder1)
mv ~/desktop/notes.txt ~/documents/ (moves a file using absolute paths)
cp Copy. Creates a duplicate of a file or folder. It also takes the item to copy and a destination as arguments. cp resume.txt resume-backup.txt (creates a copy with a new name)
cp file1 folder1/ (copies file1 into folder1)
cp -r Copy recursively. The -r option is necessary to copy an entire directory and all its contents. cp -r project/ backup-project/ (copies the project directory and everything in it)

Key Takeaways üí°

  • Single-Step Actions: Unlike the graphical interface, many commands like mv and cp accomplish a full task in a single step.
  • Command Structure: Many commands follow a common pattern: [command] [options] [arguments].
  • Tab is Your Friend: Use the Tab key to auto-complete commands and file paths, saving you from having to type everything out and helping you avoid typos.

Understanding Your Location

In a Linux environment, you don't have visual cues like a mouse to show you where you are. It's crucial to know your current location, or working directory, which is the folder you are currently "in" at the command line.

Root Directory (/)

The very top of the Linux file system is called the root directory. All other files and directories are contained within it.

Home Directory (~ or /home/user)

Every user has a home directory, which is a personal space for their files. The tilde symbol (~) is a shortcut that always represents your current user's home directory, no matter where you are in the file system.

Case Sensitivity

Linux is case-sensitive, meaning Desktop is different from desktop. Be careful with capitalization when typing file or directory names.

The Tab Key

Use the Tab key for auto-completion. This prevents typos and speeds up navigation. If multiple files or folders match what you've typed, double-tapping Tab will show you all the options.

üí° Tips for Getting Started

Opening the Terminal

You can open a terminal by right-clicking and selecting "Open in Terminal" or by searching for "terminal" in your applications menu. Be aware that how you open it may change your starting location.

Full vs. Relative Paths

You can navigate using a full path (starting from /, e.g., /home/sandbox/documents) or a relative path (starting from your current location).

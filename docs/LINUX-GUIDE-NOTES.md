# Linux Guide Notes

## Boot Menu Keys

- Emphasize repeatedly tapping the boot key as soon as the logo appears
- If you miss the window, restart and try again

Common keys:
- Dell: F12
- HP: Esc, F9
- Lenovo: F12, Novo Button
- Acer: F12
- ASUS: Esc, F8
- MSI: F11
- Toshiba: F12
- Samsung: Esc, F12

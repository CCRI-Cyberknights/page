# Planning Documentation Index

This folder contains active planning and implementation documents that are not yet consolidated into canonical production documentation.

## Folder Structure

- **`active/`** - Documents currently being planned or implemented
- **`implemented/`** - Completed features awaiting consolidation into canonical docs
- **`archived/`** - Historical reference documents (already consolidated or deprecated)

## Workflow

1. **Create Planning Doc** → Place in `active/`
2. **Feature Complete** → Move to `implemented/`
3. **Consolidated** → Move to `archived/` or delete (Git preserves history)
